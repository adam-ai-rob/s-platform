- **Feature**: s-building — admin API (`/building/admin/*`) with scoped-permission gating. All seven endpoints land per the conventions in `docs/architecture/09-api-conventions.md` — `POST /buildings`, `GET /buildings`, `GET /buildings/{id}`, `PATCH /buildings/{id}`, `POST /buildings/{id}:archive`, `POST /buildings/{id}:activate`, `DELETE /buildings/{id}`.
