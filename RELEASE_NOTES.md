- **Feature**: s-building — Building entity, repository, and service. `packages/s-building/core/src/buildings/buildings.entity.ts` declares the Zod entity with BCP-47 language, ISO 4217 currency, IANA tz, ISO 3166-1 alpha-2 country, and schema.org-style Address (with optional `location: { lat, lng }`). Cross-field invariant: `supportedLanguages` must include `primaryLanguage` (enforced on both create and update; PATCH merges and re-validates). `buildings.repository.ts` extends `BaseRepository` with `findById`, conditional `insert` (attribute_not_exists), `update`, `deleteById`, `listByStatus` via the `ByStatus` GSI (scanIndexForward false → newest first), and `scanPage` for the #68 backfill. `buildings.service.ts` is permission-agnostic and handles validation, status-transition legality (`ALLOWED_TRANSITIONS`: draft→active, active↔archived; archived→draft forbidden), timestamp management (paired ISO + `*Ms` epoch), and ULID id generation.
